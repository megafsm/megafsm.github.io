Expressions:

1) x

2) op x

3) x op y

* - where op is operator

Keywords

Description

bool, str, int, float, null, array, object

data types

true, false

expression result

len(x)

length of the string, array or object keys

lowerCase(x)

converts string value to lowercase

float(x)

casts value to float

str(x)

casts value to str

General terms[d]:

if x is str then

if lowerCase(x) in ["true","yes"] then x = true

if lowerCase(x) in ["false","no","nan","none","null"] then x = false

same with y

        When referring to object's keys or array's elements, following syntax can be used:

e.g.:

1) {"a" : {"b" : 1}} -- a.b -> 1

2) {"a" : [{"c" : 4}, {"e" : 7}, {"t": 11}]} -- a.1.e -> 7

Also note: in MegaBot and MegaFSM/Chabrum projects in conditions field you'd need to mention "data" and "global" variables as one of the root keys; e.g. data.message.text=='/start' also conditions string should be surrounded by quotation marks "

Extended expressions:

        E.g.:

((true != false) and (true or false) or 5<6)

### 1 ==> x

 -- expression checking

        if x is bool then return x

        if x is int or float then

                if x == 0 then return true

                else return false

        if x is str and len(x)>0 then return true

        else return false

        if x is null then return false

        if x is object or x is array then return true

### 2 ==> op x

        not 

 -- negative expression checking

        if x is bool then return !x

if x is null then return true

### 3 ==> x op y

==

 -- comparison

If (x and y) are "of the same type" then return x == y

        Other cases:

        if x is bool and y is in [int, float] then return x == (y != 0)

        if y is bool and x is in [int, float] then return y == ( x != 0)

        if (x and y) is in [int, float] then return float(x) == float(y)

        if x is str and y is in [int, float] then return x == str(y)

        if y is str and x is in [int, float] then return y == str(x)

!=

-- opposite comparison

        same as "==" but opposite result

>

-- comparison

        if x is int and y is int then return x > y

        if x is float and y is float then return x > y

        if x is float and y is int then return float(x) > float(y)

        if y is float and x is int then return float(x) > float(y)

        if x is in [str, array] and y is in [int, float] then return len(x) > y

        if x is str and y is in [array, str] then return len(x) > len(y)

        if x is array and y is in [array, str] then return len(x) > len(y)

        if x is bool and y is bool then return x > y *( true > false ) 

<

-- comparison

        if x is int and y is int then return x < y

        if x is float and y is float then return x < y

        if x is float and y is int then return float(x) < float(y)

        if y is float and x is int then return float(x) < float(y)

        if x is in [str, array] and y is in [int, float] then return len(x) < y

        if x is str and y is in [array, str] then return len(x) < len(y)

        if x is array and y is in [array, str] then return len(x) > len(y)

        if x is bool and y is bool then return x < y *( true > false ) 

>=

        -- comparison

if x is int and y is int then return x >= y

        if x is float and y is float then return x >= y

        if x is float and y is int then return float(x) >= float(y)

        if y is float and x is int then return float(x) >= float(y)

        if x is in [str, array] and y is in [int, float] then return len(x) >= y

        if x is str and y is in [array, str] then return len(x) >= len(y)

        if x is array and y is in [array, str] then return len(x) >= len(y)

        if x is bool and y is bool then return x >= y *( true > false )

<=

-- comparison

        if x is int and y is int then return x <= y

        if x is float and y is float then return x <= y

        if x is float and y is int then return float(x) <= float(y)

        if y is float and x is int then return float(x) <= float(y)

        if x is in [str, array] and y is in [int, float] then return len(x) <= y

        if x is str and y is in [array, str] then return len(x) <= len(y)

        if x is array and y is in [array, str] then return len(x) <= len(y)

        if x is bool and y is bool then return x <= y *( true > false )

in

 -- case insensitive "in"

        if x is str and y is in [array, dict] then lowerCase(x) in y

        if x is str and y is str then lowerCase(x) in lowerCase(y)

if x is int and y is str then str(x) in y

not in

-- case insensitive "not in"

        if x is str and y is in [array, dict] then lowerCase(x) not in y

        if x is str and y is str then lowerCase(x) not in lowerCase(y)

if x is int and y is str then str(x) not in y

$in

-- case sensitive "in"

        if x is str and y is in [array, dict] then x in y

        if x is str and y is str then x in y

if x is int and y is str then str(x) in y

not $in

-- case sensitive "not in"

        if x is str and y is in [array, dict] then x not in y

        if x is str and y is str then x not in y

if x is int and y is str then str(x) not in y

* operators "in", "not in", "$in", "not $" should be surrounded with spaces; e.g.: "a in b"

** operator "not" has only one space in the end; e.g.: "not true"
